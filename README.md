# 自學時寫的第一個網頁
`在自學以及複習網頁前端內容時，使用所學寫出的第一個頁面。 版面使用假文及假圖代表內容`

## 此網頁特色
- 僅使用HTML、CSS、jquery
- 選單有3個簡單的超連結，連結到網頁內該內容的位置
- 右下角有回到最上方的按鈕
- 利用jquery控制選單開關
- 有RWD
